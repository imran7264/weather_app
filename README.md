# 🌤️ Weather App
A clean and responsive weather application that provides real-time weather updates.

## 🚀 [Live Demo](YOUR_LINK_HERE)

## ✨ Features
- [x] Search for weather by city name
- [x] Displays temperature, humidity, and wind speed
- [x] Responsive design for mobile and desktop

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript React.js (ES6+)
- **Build Tool:** Vite
- **API:** [OpenWeatherMap API](https://openweathermap.org)

## 📦 Installation & Setup
1. Clone the repo: `git clone https://github.com`
2. Open `index.html` in your browser (or run `npm run dev` if using Vite).
Use code with caution.
